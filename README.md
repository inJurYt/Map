# Map
最终版本停车助手
hightway_parking
###主界面：地图（当前位置、显示附近车辆、附近停车场、选定目的地、导航路径（语音）、消息推送（语音）、查找栏）
###侧边栏：头像、用户名和余额（头像下方）、个人中心（用户名、余额、更改密码、更改头像、消费记录）、充值（输入金额充值、积分形式）、历史推送
###到达停车场后语音提示进入停车场模式（高速公路相同）
##停车场模式
###自动选定最近空车位，显示导航路径（实时刷新）
###到达车位后开始收费计时（语音提示），离开时停止并扣费（语音提示）
###将车主导航至其车辆
###离开停车场时切换至普通模式
##高速公路模式
###计费（积分形式） ###附近特殊事件、天气状况
